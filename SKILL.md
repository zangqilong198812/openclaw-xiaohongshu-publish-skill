# 小红书发布 Skill

## 使用前提

1. **账号信息文件**：用户目录下必须有 `~/xiaohongshu-account.md` 文件，包含账号定位、受众、风格等
2. **用户已登录小红书**：首次使用让用户手动登录一次，浏览器记住登录状态

## 账号信息格式

用户需要创建 `~/xiaohongshu-account.md`，格式如下：

```markdown
# 小红书账号信息

## 账号定位
美妆博主 / 美食博主 / 3C数码 / 生活方式 / etc.

## 目标受众
年轻女性 / 职场人士 / 学生 / etc.

## 内容风格
温柔亲切 / 专业严肃 / 幽默风趣 / etc.

## 产品/服务（可选）
要推广的产品或服务描述

## 其他要求
- 禁止出现的词汇：xxx
- 必须包含的关键词：xxx
```

## 发布历史

发布历史保存在 `~/xiaohongshu-history.md`，每次发布后自动记录：
- 发布日期时间
- 内容类型（图文/视频）
- 素材路径
- 生成的标题
- 笔记链接（如果有）

## 发布流程

### Step 1: 读取账号信息

读取用户的 `~/xiaohongshu-account.md` 文件。

### Step 2: 获取素材信息

询问用户：
- **内容类型**：图文还是视频？
- **素材路径**：图片/视频在电脑中的路径（支持多张图片，如 ~/images/1.jpg, ~/images/2.jpg）
- **素材描述**：简单描述素材内容（用于 AI 生成文案）

### Step 3: AI 生成文案

根据账号信息 + 素材描述，生成：

- **标题**：控制在 20 字以内（小红书标题限制）
- **正文**：符合账号风格，介绍素材内容
- **话题**：1-3 个相关话题，如 #xxx

生成后展示给用户确认，用户同意后再继续。

### Step 4: 打开小红书发布页

```
browser action=open targetUrl=https://creator.xiaohongshu.com/publish/publish
```

### Step 5: 上传内容

**图文上传**：
1. `browser action=snapshot` 查看页面元素
2. 点击上传按钮（使用 element ref）
3. `browser action=act` 上传文件

**视频上传**：
1. 先点击视频标签切换到视频模式
2. 上传视频文件
3. 等待上传完成

### Step 6: 填写文案

使用 `browser action=act` type 输入标题、正文、话题。

### Step 7: 发布

点击发布按钮，确认发布成功。

### Step 8: 记录发布历史

发布成功后，更新 `~/xiaohongshu-history.md`，追加本次发布记录。

## 注意事项

- 视频上传可能需要较长时间，注意处理等待
- 如果页面有验证码提示，提醒用户手动处理
- 发布前必须让用户确认 AI 生成的文案
- 标题严格控制在 20 字以内
